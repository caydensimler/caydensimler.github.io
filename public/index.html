HealthEPet

A website used to bridge the gap between pet owners and veterinarian clinics.

Built using Laravel framework, PHP, MySQL, Bootstrap, CSS, HTML, JS, and jQuery. It is an application that caters toward veterinarians that are moving from paper filing to electronic filing of shot records, medications prescribed, treatments that were provided, and gives pet owners ability to register and view their pet records instantaneously. I held the position of sole front-end developer and was able to incorporate custom modals and velocity.js animations.

Languages Used:
HTML • CSS • Javascript (jQuery) • Velocity.js • PHP • MySQL • Laravel

Styledit (Reddit Clone

A website used to create blog posts.

A reddit clone built as a solo project designed to learn Laravel. It has all the primary functions that Reddit has such as the ability to create a post and upvote/downvote posts but it contains a better aesthetic look than the barebones design that Reddit is known for.

Languages Used:
HTML • CSS • Javascript (jQuery) • PHP • MySQL • Laravel

Ad-Up (Craigslist Clone)

A website for listing items to sell.

Craigslist clone built in a team of three over the course of 5 days, using Full Stack Web Development Technologies. It also includes MySQL Databases for user signup, user login, and the ability to add/delete/create listings. I was responsible for the entire front end of the website, and assisted with the back end through creation of the functionality of user login/signup and listing creation.

Languages Used:
HTML • CSS • Javascript (jQuery) • PHP • MySQL
